# [savana](https://github.com/cortes-ciriano-lab/savana)
## 一、安装工具
### 1. 创建环境
        conda create -n savana -y
### 2. 激活环境
        conda activate savana
### 3.安装[savana](https://github.com/cortes-ciriano-lab/savana#installation)
* 根据官方指南，直接用conda安装

        conda install -c bioconda savana
### 4.验证安装

        savana --version
